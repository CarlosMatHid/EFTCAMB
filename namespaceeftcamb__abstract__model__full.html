Detailed Description

This module contains the abstract definition of all the places where EFTCAMB interacts with CAMB in case of a full mapping model. All models implementing a model in which the cosmological background is computed from the values of the EFT functions should inherit from this class.

Author
Bin Hu, Marco Raveri
