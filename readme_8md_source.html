29 ### 8. EFTCAMB source files:
30 
31 In the folder ``eftcamb`` all the source files for EFTCAMB are stored.
32 In an effort to have small and readable files the the naming convention allows to have an
33 intuition of the hierarchy of the code from alphabetical order of files.
34 
35 For this reason we use the following convention for the prefixes:
36 
37 * ``01_`` compile time utilities
38 * ``02_`` pure algorithms
39 * ``03_`` general parametrizations for 1D functions
40 * ``04_`` general parametrizations for 2D functions
41 * ``05_`` abstract implementation of EFT models
42 * ``06_`` implementation of pure EFT models
43 * ``07_`` implementation of alternative EFT parametrizations
44 * ``08_`` implementation of designer mapping EFT models
45 * ``09_`` implementation of full mapping EFT models
46 * ``10_`` general EFT algorithms (RGR, stability, init)
