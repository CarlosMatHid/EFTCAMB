8 ### 1. EFTCAMB Requirements:
9 
10 Compiling EFTCAMB requires a modern fortran compiler capable of handeling F2008 features.
11 These includes:
12 
13  ifort v>15.0 (?)
14  gcc/gfortran v>6.0
15 
16 To use other parts of the code, like the test or documentation parts other requirements have to be met.
17 These include a fully fledged python installation. We warmly suggest to install a
18 bundled package like (https://www.continuum.io/downloads).
19 
20 A docker with all the required libraries is available at [dockerhub](https://hub.docker.com/r/eftcamb/eftbox/).

64 ### 8. EFTCAMB source files:
65 
66 In the folder ``eftcamb`` all the source files for EFTCAMB are stored.
67 In an effort to have small and readable files the the naming convention allows to have an
68 intuition of the hierarchy of the code from alphabetical order of files.
69 
70 For this reason we use the following convention for the prefixes:
71 
72 * ``01_`` compile time utilities
73 * ``02_`` pure algorithms
74 * ``03_`` EFTCAMB cache containing the storage for all cosmological quantities of interest
75 * ``04_`` general parametrizations for 1D functions
76 * ``05_`` general parametrizations for 2D functions
77 * ``06_`` abstract implementation of EFT models
78 * ``07_`` implementation of pure EFT models
79 * ``08_`` implementation of alternative EFT parametrizations
80 * ``09_`` implementation of designer mapping EFT models
81 * ``10_`` implementation of full mapping EFT models
82 * ``11_`` general EFT algorithms (RGR, stability, init)
