Detailed Description

This module contains the definition of a time step cache for EFTCAMB. To perform EFTCAMB calculations, at any timestep, we need several quantities. Densities, values of the EFT functions and so on. Instead of complicating the interface of the functions we shall define a type that holds all these informations. We also need a cache for cosmological parameters that are used in the calculations.
